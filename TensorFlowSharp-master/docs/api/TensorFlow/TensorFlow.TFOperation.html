Class TFOperation

Represents a computation node in the graph. Tensorflow operations are attached to a Tensorflow.TFGraph.

Inheritance
System.Object
TFOperation
Namespace: TensorFlow
Assembly: TensorFlowSharp.dll
Syntax
public class TFOperation
Remarks

TFOperations are usually created by invoking one of the methods in Tensorflow.TFGraph, but they can also be constructed manually using the low-level Tensorflow.TFOperationDesc API.
