Enum TFDataType

The data type for a specific tensor.

Namespace: TensorFlow
Assembly: TensorFlowSharp.dll
Syntax
public enum TFDataType
Remarks

Tensors have uniform data types, all the elements of the tensor are of this type and they dictate how TensorFlow will treat the data stored.
